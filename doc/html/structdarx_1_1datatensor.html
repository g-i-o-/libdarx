Detailed Description

Low-level structure representing a tensor of data.

A tensor is a (possibly) multi-dimmensional array of data, with each element of the tensor belong to some datatype. This tensor has an associated name, rank (it's dimmensionality), and size (in all it's dimmensions). The data may be compressed using one of the available algorithms (none at the moment).
